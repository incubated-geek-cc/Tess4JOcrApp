# Tess4JOcrApp
## A native Desktop application built with [Tess4J](http://tess4j.sourceforge.net/) in Java8
### Uses the open-sourced [Tesseract OCR Engine](https://github.com/tesseract-ocr/tesseract) & JNA Java Wrapper Class
